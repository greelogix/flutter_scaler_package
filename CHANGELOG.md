## 1.0.0

* Initial stable release of FlutterAdaptiveScaler.
* Automatic responsive scaling for fonts, sizes, padding, icons, and radius.
* Support for runtime orientation changes and window resizing (Web/Desktop).
* Easy-to-use extensions on `num` (`.w`, `.h`, `.sp`, `.r`, `.i`, `.p`).
* `FlutterScaler` wrapper widget with `builder` pattern for full reactivity.
* Example application demonstrating a professional responsive dashboard.
* Published by GreeLogix.
